2. Sintaxe básica:

Um programa Java consiste em classes. Você deve criar uma classe com um método main para executar o código. Aqui está um exemplo simples:

public class MeuPrograma {
  public static void main(String[] args) {
   System.out.println("Olá, Mundo!");
  }
}

Linhas de código em Java terminam com ponto e vírgula (;).
