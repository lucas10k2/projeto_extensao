01 Seletores CSS:

Os seletores CSS são usados para identificar os elementos HTML que você deseja estilizar. Eles podem ser baseados em tags, classes, IDs ou outros atributos. Aqui estão alguns exemplos de seletores CSS:


Seletor de tag: p (estiliza todos os parágrafos na página).


Seletor de classe: .destaque (estiliza elementos com a classe "destaque").


Seletor de ID: #cabecalho (estiliza o elemento com o ID "cabecalho").
