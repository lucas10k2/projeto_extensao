Atividade Prática: Estilizando Tags HTML com CSS

Dentro da tag <style>, adicione um seletor para a tag <h1> e defina a propriedade color como "red" para tornar o texto vermelho:

h1 {
color: red;
}

Abaixo, adicione um seletor para a tag <p> e defina a propriedade color como "purple" para tornar o texto roxo:

p {
color: purple;
}
