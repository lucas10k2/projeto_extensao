01 Tags HTML:

<!DOCTYPE html>: Define o tipo de documento.

<html>: Marca o início de um documento HTML.

<head>: Contém informações sobre o documento, como metadados e links para folhas de estilo.

 <title>: Define o título da página que aparece na barra de título do navegador.

<body>: Contém o conteúdo visível da página, como texto, imagens e links.

<header>: Define o conteúdo de um cabeçalho.

<footer>: Define o conteúdo de um rodapé.

<nav>: Define área com links para navegação.

<main>: Container com conteúdo principal.

<section>: Define uma seção.

<h1>: Define um título, tendo variação até o h6 com pesos menores.

<p>: Define um parágrafo de texto.
