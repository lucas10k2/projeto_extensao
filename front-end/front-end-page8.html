05 Atividade Prática:

Criando sua Primeira Página HTML

Objetivo:

Criar uma página HTML simples que inclua cabeçalhos,rodapés, titulos,paragrafos.

Instruções:

  1. Abra um editor de código HTML de sua escolha (recomendamos usar um editor de código como o Visual Studio Code).

  2. Crie um novo documento HTML e adicione a estrutura básica.

  3. Dentro do <body>, crie um cabeçalho principal <h1> com o texto "Bem-vindo à Minha Página HTML".

  4. Abaixo do cabeçalho, crie um parágrafo <p> com uma breve descrição sobre o que a página tratará. Por exemplo, "Nesta página, você aprenderá os conceitos básicos do HTML."

  5. Certifique-se de que sua página HTML esteja corretamente estruturada e todas as tags estejam fechadas.

  6. Salve o arquivo com uma extensão ".html" (por exemplo, "minha_pagina.html").

  7. Abra o arquivo em um navegador da web para verificar se sua página HTML está funcionando corretamente.
